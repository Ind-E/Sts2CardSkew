Adds the skew effect from Balatro to cards and allows you to enable foil, polychrome, holo, or negative shaders per-card.

## Installation

Extract the .zip from the [latest release](https://github.com/Ind-E/BalatroEffects/releases/latest) into your Slay the Spire 2 mods folder
- Windows and Linux: `Steam/steamapps/common/Slay the Spire 2/mods/`
- Mac: `SlayTheSpire2.app/Contents/MacOS/mods`
